# Clear-PsNetDnsSearchSuffix

## SYNOPSIS
Clear-PsNetDnsSearchSuffix

## SYNTAX

```
Clear-PsNetDnsSearchSuffix [<CommonParameters>]
```

## DESCRIPTION
Running this command with elevated privilege.
Remove all entries from the DnsSearchSuffixList

## EXAMPLES

### EXAMPLE 1
```
Clear-PsNetDnsSearchSuffix
```
